Usage of CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It allows you to control the layout, colors, fonts, and overall appearance of your web pages. Here are some common ways to use CSS:

4. CSS Selectors

CSS selectors are used to target specific HTML elements to apply styles. Some common selectors include:

  • Universal Selector: Targets all elements on the page. Example: * { margin: 0; }

  • tag Selector: Targets all elements of a specific type. Example: p { color: red; }

  • Group Selector: Targets multiple elements. Example: h1, h2, h3 { font-family: Arial; }

  • Element Selector: Targets all elements of a specific type. Example: p { color: red; }

  • Class Selector: Targets elements with a specific class attribute. Example: .my-class { font-size: 16px; }

  • ID Selector: Targets a single element with a specific id attribute. Example: #my-id { background-color: yellow; }

  • Decendent selector: Targets elements that are descendants of a specified element. Example: div p { color: blue; }

  • Attribute Selector: Targets elements based on their attributes. Example: input[type="text"] { border: 1px solid black; }

5. CSS Properties

CSS properties are used to define the styles for the selected elements. Some common properties include:

  • Color: Sets the text color. Example: color: red;

  • Font-size: Sets the size of the text. Example: font-size: 16px;

  • Background-color: Sets the background color of an element. Example: background-color: yellow;

  • Margin: Sets the space around elements. Example: margin: 10px;

  • Padding: Sets the space between the content and the border of an element. Example: padding: 5px;

7.CSS units

CSS units are used to specify the size of elements, fonts, and other properties. Some common CSS units include:

  • Absolute Units: These units are fixed and do not change based on the context. Examples include inches (in), centimeters (cm), millimeters (mm), points (pt), and picas (pc). For example: width: 2in;

  • Relative Units: These units are based on the size of the parent element or the root element. Examples include em, rem, and percentage (%). For example: font-size: 1.5em;

  • Pixels (px): A fixed unit that represents a specific number of pixels. Example: font-siz
    e: 16px;

  • Em (em): A relative unit that is based on the font size of the parent element. Example: font-size: 1.5em;

  • Rem (rem): A relative unit that is based on the font size of the root(HTML) element. Example: font-size: 1.5rem;

  • Percentage (%): A relative unit that is based on the size of the parent element. Example: width: 50%;

  • Viewport Units: These units are based on the size of the viewport (the visible area of the web page). Examples include vw (viewport width) and vh (viewport height). For example: width: 50vw;

  • Viewport Height (vh): A relative unit that is based on the height of the viewport. Example: height: 50vh;

  • Viewport Width (vw): A relative unit that is based on the width of the viewport. Example: width: 50vw;

  • Viewport Minimum (vmin): A relative unit that is based on the smaller dimension of the viewport. Example: font-size: 2vmin;

  • Viewport Maximum (vmax): A relative unit that is based on the larger dimension of the viewport. Example: font-size: 2vmax;

  • Ch (ch): A relative unit that is based on the width of the "0" character in the current font. Example: width: 10ch;

  • Ex (ex): A relative unit that is based on the height of the "x" character in the current font. Example: font-size: 2ex;

8.CSS Box Model

The CSS box model is a fundamental concept in web design that describes how elements are structured and how they interact with each other. It consists of four main components:

  • Content: The innermost part of the box that contains the actual content of the element, such as text or images.

  • Padding: The space between the content and the border. It can be used to create space around the content within the element.

  • Border: The area that surrounds the padding and content. It can be styled with different colors, widths, and styles (e.g., solid, dashed, dotted).

  • Margin: The outermost part of the box that creates space between the element and other elements on the page. It can be used to control the spacing between elements.

Pseudo Class

A pseudo-class is a keyword added to selectors that specifies a special state of the selected elements. It allows you to apply styles to elements based on their state or position in the document. Some common pseudo-classes include:

  • :hover - Applies styles when the user hovers over an element. Example: a:hover { color: red; }

  • :active - Applies styles when an element is being activated (e.g., clicked). Example: button:active { background-color: blue; }

  • :focus - Applies styles when an element has focus (e.g., when a user clicks on an input field). Example: input:focus { border-color: green; }

  • :nth-child(n) - Applies styles to the nth child of a parent element. Example: li:nth-child(2) { color: orange; }

  • :first-child - Applies styles to the first child of a parent element. Example: p:first-child { font-weight: bold; }

  • :last-child - Applies styles to the last child of a parent element. Example: p:last-child { font-style: italic; }

  • :not(selector) - Applies styles to elements that do not match the specified selector. Example: div:not(.highlight) { background-color: lightgray; }

Pseudo Element

A pseudo-element is a keyword added to selectors that allows you to style specific parts of an element. It is used to create and style elements that do not exist in the HTML document. Some common pseudo-elements include:

  • ::before - Inserts content before the content of an element. Example: p::before { content: "Note: "; font-weight: bold; }

  • ::after - Inserts content after the content of an element. Example: p::after { content: " (end of paragraph)"; font-style: italic; }

  • ::first-letter - Styles the first letter of an element. Example: p::first-letter { font-size: 200%; color: red; }

  • ::first-line - Styles the first line of an element. Example: p::first-line { font-weight: bold; }
